# Mauro Pires Advogados — Sistema de Gestão Financeira

Sistema interno de gestão financeira, honorários e contratos.

## Acesso
👉 **[Clique aqui para abrir o sistema](https://SEU_USUARIO.github.io/mauro-pires-gestao/)**

## Como usar
1. Abra o link acima no Chrome ou Edge
2. Os dados são salvos automaticamente no seu navegador
3. Use o botão **🗄 Backup** semanalmente para não perder dados
4. Para transferir dados entre computadores: exporte o backup em um e importe no outro

## Importante
- Cada pessoa tem seus próprios dados no seu navegador
- Para compartilhar dados entre sócios, use a função Backup (exportar/importar)
